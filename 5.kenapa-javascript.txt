alasan memilih javascript
- gratis, tidak perlu membeli software.
- hanya butuh web browser.
- tidak perlu melakukan langkah tambahan untuk kofigurasi

javascript adalah bahasa high level language(bahasa yang ekat dengan manusia)

scripting language, javascript dibuat untuk berinteraksi dengan web.

javascript adalah categiry untyped. untyped adalah bahasa pemrograman yang bisa menggunakan berbagai macam variable.

javascript adalah bhs pemrograman yg harus dijalankan dengan interpreted.

javascript = easy to learn, hard to master.
